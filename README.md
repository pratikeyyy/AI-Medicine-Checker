## 🚀 Overview

AI Medicine Checker helps users identify medicines from an uploaded image using **Google Gemini Vision AI**. The application extracts information about the medicine and presents it in a clean and user-friendly interface.

---

## ✨ Features

- 📷 Upload medicine strip or bottle images
- 🤖 AI-powered medicine identification
- 💊 Displays medicine details
- ⚡ Fast Gradio interface
- 🌐 Easy to deploy on Hugging Face Spaces

---

## 🛠 Tech Stack

- Python
- Gradio
- Google Gemini API
- Pillow
